Billie Eilish
Web animation

Bekijk de animaties!

Voor web animatie heb ik de album cover van Billie Eilish uitgekozen, omdat ik dit een hele gave foto vond. Ik dacht dat ik hier wel veel mee kon doen maar ook was het een uitdaging om bijpassende animaties hierbij te verzinnen. Als eerst heb ik de foto nagemaakt in illustrator en deze als SVG gexporteerd. Hierna heb ik deze in Brackets geanimeerd door middel van CSS en javascript.
Volgende keer zou ik de bewegingen meer easing geven zodat het er wat soepeler uitziet. Ook zou ik de illustraties beter afwerken, ook de delen die je niet ziet. Als je deze namelijk animeert kan het zijn dat je het deel wel ziet en anders ziet het er niet professioneel uit.
