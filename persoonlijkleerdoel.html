Bekijk de prototype hier!

Mijn persoonlijke leerdoel was het ontwerpen van een Interface met de focus op microinteracties en transities. Dit wilde ik graag leren in een prototype tool als Sketch of Adobe XD. Ook dacht ik dat ik ook meerdere vakken van de minor hierop kon toepassen, dus dit vond ik een leuke uitdaging.
Ik heb een interface voor de telefoon ontworpen waarbij de gebruiker door middel van een stapsgewijze filter de perfecte longboard kan vinden. Tussen deze stappen zitten mooie en orginele transities.
